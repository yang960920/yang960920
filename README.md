## 🙋‍♂️ About Me
**"고객 만족 전략가에서, 실질적인 문제를 코드로 해결하는 백엔드 개발자로!"**

과거 글로벌 OTT 기업과 F&B 브랜드에서 고객 경험(CX)을 개선하고 시스템을 효율화했던 경험을 바탕으로, 이제는 **코드를 통해 비즈니스 로직과 사내 프로세스를 혁신**하고 있습니다. 단순히 돌아가는 코드를 넘어서, '사용자가 진짜 필요로 하는 것'이 무엇인지 고민합니다.

- 🚀 **실무 밀착형 문제 해결:** Google Apps Script를 활용해 배차 일지, 자산 관리, 일일 업무 보고 등 사내 비효율을 걷어내는 맞춤형 웹 어플리케이션을 기획하고 개발합니다.
- 💡 **리더십 & 커뮤니케이션:** Java 백엔드 팀 프로젝트 리더로서, 위기 상황을 투명한 소통과 멘토링으로 극복하여 우수상을 이끌어낸 경험이 있습니다.
- 🌐 **유연한 배포와 인프라:** Vercel 환경에서의 다중 페이지 커뮤니티 및 기업 홈페이지 구축 경험을 통해 프론트엔드 배포와 DB 연동 등 폭넓은 시야를 갖추고 있습니다.
